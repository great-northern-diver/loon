l_plot is a generic function for creating interactive 

  visualization environments for R objects.
Parameters
xthe x coordinates of points in the l_plot. Alternatively, a single plotting structure, function, or any R object having an l_plot method can be provided.
ythe y coordinates of points in the l_plot, optional if x is an appropriate structure.
Returns
widget handle
Examples
p1 = l_plot(iris['Sepal.Length'],iris['Sepal.Width'],color = iris['Species'],title = "First plot")
# The names of the info states that can be
# accessed or set. They can also be given values as
# arguments to l_plot.default()
p1.names
p1["size"] <- 10
