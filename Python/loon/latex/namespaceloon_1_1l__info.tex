Retrieve Information about the States of a Loon Widget. 

\begin{DoxyParagraph}{Description}
Loon\textquotesingle{}s built-\/in object documentation. Can be used with every loon object that has plot states including plots, layers, navigators, contexts. This is a generic function.
\end{DoxyParagraph}

\begin{DoxyParams}{Parameters}
{\em target} & target loon class \\
\hline
{\em states} & list with names of states. \mbox{[}\textquotesingle{}all\textquotesingle{}\mbox{]} is treated as a keyword and results in returning information on all plot states \\
\hline
\end{DoxyParams}
\begin{DoxyReturn}{Returns}
a named nested list with one element per state. The list elements are also named lists with type, dimension, defaultvalue, and description elements containing the respective information.
\end{DoxyReturn}
