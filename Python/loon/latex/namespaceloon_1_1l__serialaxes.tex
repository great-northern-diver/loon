Create a Serialaxes Widget. 

\begin{DoxyVerb}  The seerialaxes widget displays multivariate data either as a stacked star glyph plot, or as a parallel coordinate plot.
\end{DoxyVerb}

\begin{DoxyParagraph}{Details}
The scaling state defines how the data is scaled. The axes display 0 at one end and 1 at the other. For the following explanation assume that the data is in a nxp dimensional matrix. The scaling options are then \tabulinesep=1mm
\begin{longtabu}spread 0pt [c]{*{2}{|X[-1]}|}
\hline
variable &per column scaling \\\cline{1-2}
observation &per row scaling \\\cline{1-2}
data &whole matrix scaling \\\cline{1-2}
none &do not scale \\\cline{1-2}
\end{longtabu}
