Detailed Description

Use Map with zoom 1

  • Viewport is such that it has the same aspect ratio like the activewidget.
  • The deltaX and deltaY need to be kept in sync with all layers
  • Outline for all layered elemets
  • Use ...WorldviewVisuals if implemented
  • Scatterplot Outline data points Use Worldview Visuals instead of ordinary Visuals if they exist. The Scatterplot layer respective worldview uses always Keep the deltaX and deltaY updated to be the range of all
