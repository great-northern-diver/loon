5  Loon is a toolkit for highly interactive data visualization. Interactions with plots are provided with mouse and keyboard gestures as well as via command line control and with inspectors that provide graphical user interfaces (GUIs) for modifying and overseeing plots.
6 
7 Details:
8  Currently, loon implements the following statistical graphs: histogram,
9  scatterplot, serialaxes plot (star glyphs, parallel coordinates)
10  and a graph display for creating navigation graphs.
11 
12  Some of the implemented scatterplot features, for example,
13  are zooming, panning, selection and moving of points, dynamic
14  linking of plots, layering of visual information such as maps
15  and regression lines, custom point glyphs (images, text, star glyphs),
16  and event bindings. Event bindings provide hooks to evaluate custom code
17  at specific plot state changes or mouse and keyboard interactions. Hence,
18  event bindings can be used to add to or modify the default behavior of the
19  plot widgets.
20 
21  Loon's capabilities are very useful for statistical analysis tasks
22  such as interactive exploratory data analysis, sensitivity analysis,
23  animation, teaching, and creating new graphical user interfaces.
