15  Details:
16  Loon's standard linking model is based on two states,
17  `linkingGroup` and `linkingKey`. The full capabilities of the
18  standard linking model are described here. However, setting the
19  `linkingGroup` states for two or more displays to the same string is
20  generally all that is needed for linking displays that plot data from the
21  same data frame. Changing the linking group of a display is also the only
22  linking-related action available on the analysis inspectors.
23 
24  The first linking level is as follows: loon's displays are linked if they
25  share the same string in their `linkingGroup` state. The default
26  linking group 'none' is a keyword and leaves a display un-linked.
27 
28  The second linking level is as follows. All `n`-dimensional states can
29  be linked between displays. We call these states *linkable. Further,
30  only linkable states with the same name can be linked between displays. One
31  consequence of this *shared state name* rule is that, with the
32  standard linking model, the `linewidth` state of a serialaxes display
33  cannot be linked with the `size` state of a scatterplot display. Also,
34  each display maintains a list that defines which of its linkable states
35  should be used for linking; we call these states the *used linkable*
36  states. The default used linkable states are as follows
37  <table>
38  <tr><th>Display <th> Default *used linkable* states
39  <tr><td>scatterplot <td> `selected`,`color`, `active`, `size`
40  <tr><td>histogram <td> `selected`,`color`, `active`
41  <tr><td>serialaxes <td> `selected`,`color`, `active`
42  <tr><td>graph <td> `selected`,`color`, `active`, `size`
43  </table>
44 
45  If any two displays are set to be linked (i.e. they share the same linking
46  group) then the intersection of their *used linkable* states are
47  actually linked.
48 
49  The third linking level is as follows. Every display has a
50  `n`-dimensional `linkingKey` state. Hence, every data point has
51  an associated linking key. Data points between linked plots are linked if
52  they share the same linking key.
53
