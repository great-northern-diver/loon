10 # #' @param states a character vector of the states to be copied. If `NULL` (the default),
11 # #' then all states in common (excluding those identified by exclusion parameters)
12 # #' are copied from the `source` to the `target`.
13 # #' @param exclude a character vector naming those common states to be excluded
14 # #' from copying. Default is NULL.
15 # #' @param excludeBasicStates a logical indicating whether certain basic states
16 # #' are to be excluded from the copy (if `TRUE`, the default).
17 # #' These states include those derived from data variables (like
18 # #' "x", "xTemp", "zoomX", "panX", "deltaX", "xlabel", and the "y" counterparts)
19 # #' since these values determine coordinates in the plot and so are typically not to be copied.
20 # #' Similarly "swapAxes" is one of these basic states because in \code{l_compound} plots such
21 # #' as \code{l_pairs()} swapping axes can wreak havoc if unintended.
22 # #' Finally, an important pair of basic states to exclude are
23 # #' "linkingKey" and "linkingGroup" since such changes require proper
24 # #' synchronization.
25 # #'
26 # #' Setting `excludeBasicStates = TRUE` is a simple way to avoid copying the values of these
27 # #' basic states.
28 # #' Setting `excludeBasicStates = FALSE` will allow these to be copied as well.
29 # #' @param returnNames a logical to indicate whether to return the names
30 # #' of all states successfully copied for all plots. Default is `FALSE`
31 # #'
32 # #' @return a character vector of the names of the states successfully copied
33 # #' (for each plot whose states were affected), or
34 # #' NULL if none were copied or `returnNames == FALSE`.
35 # #'
